FiltEn

enhanced search filter

Requirements

Since the script scans all the results on the page, pagination must be disabled. Otherwise the filters would only reflect the items on the current page of the pager and would not be a correct reflection of the items.

Since the script hides filtered results, results should be laid out in a responsive manner. Tabular layouts do not work well because each item is “locked” into its original row and column. As items are hidden, the remaining items do not shift to fill in the gaps. A responsive layout is more conducive to this approach because it can react to changes in item visibility due to filtering.

Dependencies

FiltEn requires the following javascript libraries:
  • jQuery
  • KnockoutJS

Installation

JavaScript Libraries

Include the following javascript libraries:

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
<script src="//cdnjs.cloudflare.com/ajax/libs/knockout/3.1.0/knockout-min.js"></script>
<script src="//s3.amazonaws.com/cdn.aquilats.com/libs/filten/latest/filten.min.js" defer></script>

Stylesheet

Include the following stylesheets:

<link href="//s3.amazonaws.com/cdn.aquilats.com/libs/filten/latest/filten.min.css" rel="stylesheet" type="text/css" />

Need something a little different, you can roll your own!

Result Page HTML

Insert the following HTML into the item search result page:

<div class="filten"></div>

Item Metadata

Add the class “filten-item” to each item's containing HTML element. In the same element, add the following attributes to specify search criteria values for the item:

data-filten-CRITERIA=”VALUE”

Replace CRITERIA with the criteria name and VALUE with the item's value for that criteria. You may have as many of these as you require. Not all items have to have every criteria, so it’s ok to omit the attribute for those items. Use the preProcess option to sanitize value prior to using them as search criteria. <!-- HTML comments --> and trailing/leading whitespaces are automatically removed from string values.

Advanced Options

Some criteria may require more than just a name/value pair to achieve the desired effect. To specify additional options for specific criteria, set the filten.options varialbe within <script> tags below the filten.min.js inclusion. The variable must be an associative array whose keys correspond to the names (lowercase) of the criteria found in the item metadata. The value for each criteria name key is another object containing any of the following available options:

<script language="javascript" type="text/javascript">
  filten.options = {
    // specify additional criteria options here
  };
</script>
  • displayName : string
  • preProcess : function(value) => obj (runs after delimeter)
  • formatValue : function(value) => string
  • asRange : true | false
  • valueSort : function(a,b) => number ( return < 0 => a before b, return = 0 => equal, return > 1 => b before a)
  • delimeter : character
  • criteriaSortOrder : number (it’s ok to skip numbers – the lower this number, the earlier it will appear in the list)

Example:

var filtenCriteriaOptions = {
    dollaramount : {
        formatValue : function(value) {return "$" + value;},
        displayName : "Price",
        asRange : true,
        criteriaSortOrder : 1,
        preProcess : function(value) {
          if(typeof value == 'string')
            return value.replace(/^\s*\$/, "").trim();
          else
            return value;
        }
    },
    categories : {
        delimeter : ",",
        valueSort : function(a,b){return b-a;} 
    },
    weight : {
        formatValue : function(value) {return value + “ lbs”;},
        valueSort : function(a,b){return b-a;} 
    }
};

Styles

The script comes with a default style template in the form of a CSS file. Modifying the style of the ‘Filter Results’ box simply requires overriding the default styles with your own custom CSS. Modifying the default template is not necessary. Rather, custom styles can be added to your website’s existing CSS which will override the provided default styles.

The relevant CSS classes are identified here:

  • .filten
  • .filten-header
  • .filten-title
  • .filten-summary
  • .filten-summary-counts
  • .filten-crit
  • .filten-crit-header
  • .filten-crit-value
  • .filten-crit-value.filten-active
  • .filten-crit-value-checkbox
  • .filten-crit-value-count
  • .filten-item-filtered
